# Sora2 プロンプト: シネマティック・叙情詩 - 2月の雪と梅

**テーマ**: 冬の終わりと春の始まり、雪と梅と女性の哀愁
**長さ**: 15秒
**アスペクト比**: 16:9（横長）

## コンセプト

小津安二郎的な静謐さと黒澤明的な映画美を融合させた叙情的な映像作品。2月の日本の風景を舞台に、残雪と早咲きの梅が共存する季節の移ろいを、ゆっくりとしたドリーショットとラックフォーカスで詩的に描く。若い女性の悲しげな表情を通して、別れと新しい始まりへの複雑な感情を静かに表現する。柔らかい自然光とフィルムグレインによる映画的質感が、時間の流れと記憶の儚さを感じさせる。

---

## 構成案（シークエンス）

### 00:00 - 00:05: オープニング - 冬の名残りの風景

**シーン内容**: 雪が薄く残る日本庭園の広い風景からスタート。遠景に梅の木が見える。カメラはゆっくりと前進（ドリーイン）し、梅の木へと近づいていく。

**視覚的詳細**:
- **カメラ**: ワイドショットから開始、ゆっくりとしたドリーイン（前進）、滑らかな移動
- **ライティング**: 柔らかい自然光、曇天の拡散光で影が薄く優しい雰囲気
- **被写体**: 雪が部分的に残る地面、遠くに見える梅の木、日本庭園の静かな佇まい
- **技術**: 深い被写界深度で全体にピントが合っている状態

### 00:05 - 00:10: 展開 - 梅の花への焦点

**シーン内容**: カメラが梅の木に近づき、ラックフォーカス（ピント送り）によって梅の花にピントを移す。白とピンクの花びらが風に優しく揺れ、背景の雪景色がボケていく。

**視覚的詳細**:
- **カメラ**: ドリーイン継続、ミディアムショットからクローズアップへ移行
- **ライティング**: 自然光が花びらを柔らかく照らす、繊細な陰影
- **被写体**: 梅の花びら（白とピンクの混在）、細い枝、風による微細な動き
- **技術**: ラックフォーカスで浅い被写界深度へ移行、美しいボケ味（シネマティック・ボケ）、背景の雪景色が柔らかくぼやける

### 00:10 - 00:15: クライマックス - 女性の哀愁

**シーン内容**: 梅の花からカットして、若い女性の横顔のクローズアップへ。彼女は少し悲しげな表情で、画面の外（梅の方向）を静かに見つめている。風が彼女の髪を優しく揺らす。

**視覚的詳細**:
- **カメラ**: クローズアップ、静かで固定されたフレーム、女性の横顔を美しく捉える
- **ライティング**: サイドライトで女性の輪郭を柔らかく際立たせる、自然光による繊細な陰影
- **被写体**: 若い日本人女性の横顔、憂いを帯びた瞳、わずかに下がった口元、風に揺れる髪
- **技術**: 浅い被写界深度、背景の梅の木と雪景色が美しくボケている、女性の表情に焦点

## 生成用プロンプト（日本語参考）

```
日本の晩冬の憂いある美しさを捉えた映画的な3シーン構成、合計15秒、ワイドスクリーン16:9アスペクト比。

シーン1（0-5秒）：地面に雪が部分的に残る伝統的な日本庭園のワイド確立ショット、遠くに一本の梅の木が見える。ゆっくりとしたドリーイン移動で木に向かって滑らかに進む。曇天の下での柔らかい自然光が拡散した優しい照明を作り、影は最小限。深い被写界深度で風景全体にピントが合っている。静謐で瞑想的な雰囲気。

シーン2（5-10秒）：梅の木に近づくドリーイン移動を継続し、ミディアムショットからクローズアップへ移行。ラックフォーカス技法で背景から梅の花へピントを移す - 白とピンクの繊細な花びらが風に優しく揺れる。浅い被写界深度が美しいシネマティックボケを作り出し、雪景色の背景が柔らかくぼやける。自然光が花びらを照らし、繊細な陰影を作る。

シーン3（10-15秒）：若い日本人女性の横顔の親密なクローズアップへカット、横から捉える。彼女は画面外の梅の花の方を静かに悲しげな表情で見つめる - 憂鬱な瞳、わずかに下がった唇。優しい風が彼女の髪を柔らかく揺らす。柔らかいサイドライトが彼女のシルエットに繊細なリムライトを作る。浅い被写界深度で梅の木と雪景色が背景で美しくぼやける。固定カメラ、静的で瞑想的なフレーミング。

フォトリアリスティック、小津安二郎の静謐さと黒澤明の視覚詩にインスパイアされた映画的構図。繊細なフィルムグレインの質感、ミュートトーンのプロフェッショナルなカラーグレーディング、柔らかい自然なパレット。叙情的で憂鬱なムード、ゆっくりとしたペーシング、瞑想的な雰囲気。8K解像度、高精細、日本映画美学。
```

## 技術仕様

- **解像度**: 8K、高精細
- **アスペクト比**: 16:9（横長、標準的なワイドスクリーン）
- **長さ**: 15秒
- **カメラ技術**: ドリーイン、ラックフォーカス、固定カメラ
- **被写界深度**: 深い→浅い→浅い（シーン進行に応じて変化）
- **ライティング**: 自然光、曇天の拡散光、サイドライト
- **ムード**: 叙情的、憂鬱、瞑想的、静謐
- **スタイル**: フォトリアリスティック、日本映画美学、フィルムグレイン

## 制作ノート

このプロンプトは、日本映画の巨匠たちの美学を現代の映像生成技術で再現することを目指しています。特に：

- **小津安二郎**: 固定カメラ、静謐な構図、日常の中の叙情
- **黒澤明**: ドラマティックな自然描写、感情と風景の融合

3つのシーンは、「冬の終わり→春の兆し→人間の感情」という時間的・感情的な流れを形成し、15秒という短い時間の中で完結した物語性を持たせています。
