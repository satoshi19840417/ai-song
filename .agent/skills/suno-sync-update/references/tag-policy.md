# タグポリシー

## データソース優先順位

1. `display_tags` (Suno API レスポンス)
2. `metadata.tags` (Suno API レスポンス)
3. 既存の `tags` フィールド（スクリプトによる変更なし）

## タグ形式

- frontmatter: `tags: [タグA, タグB, タグC]`（配列形式）
- `_index.md` 表示: 先頭2件のみ（可読性優先）

## ジャンル / テーマ分類（タグ索引用）

`sync_suno_index.py` 内でタグを自動分類する際の基準：

### ジャンルと判断するキーワード（部分一致）

```
pop, rock, ballad, バラード, folk, electro, jazz, blues, クール
```

### 上記以外はテーマとして分類

感謝, 愛情, AI, テクノロジー, 冬, 雪, 応援, ポジティブ, ユーモア,
失恋, 孤独, 旅, 自由, 再会, 友情, 自己探求, 成長, デジタルデトックス など

## 注意

- タグ値はSunoのAPIから取得した英語・日本語混在でも保持する
- タグの正規化（統一）は手動で行う（スクリプトは自動変換しない）
