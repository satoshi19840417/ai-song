# YouTube Metadata Skill

YouTubeショート動画用のタイトル・概要欄を生成するスキル。

## Workflow

1. **入力情報の収集**
   - 歌詞ファイル（`.lrc`）のパス
   - 曲名
   - 歌手名

2. **歌詞ファイルの解析**
   - `.lrc`ファイルを読み込み
   - 印象的なフレーズを抽出（サビ、ラスト部分など）

3. **タイトル生成（2案）**
   - **案A（感情訴求型）**: 歌詞の印象的な一節を引用 + 曲名 / 歌手名 #shorts
   - **案B（共感型）**: ターゲット層への呼びかけ + 曲名 - 歌手名 #shorts

4. **概要欄生成（2パターン）**
   - **シンプル版**: 曲名、テーマ要約、ハッシュタグ5個、CTA
   - **詳細版**: 曲名、テーマ詳細、アーティスト情報、ハッシュタグ5個、CTA

5. **出力**
   - マークダウンファイルとして `[曲名]_YouTube概要.md` を出力
   - 出力先: 動画ファイルと同じディレクトリ

## Best Practices

### タイトル
- 曲名を必ず含める（検索性向上）
- 歌詞の一節を「」で囲んで引用（共感・クリック率UP）
- `#shorts` タグを末尾に付与

### ハッシュタグ
- **5個前後に厳選**（多すぎは逆効果）
- 必須: 曲名、歌手名、リリックビデオ
- 推奨: ジャンル、雰囲気（応援ソング、邦楽など）

### CTA（Call To Action）
概要欄の最後に1行追加：
- `▶ フル版は固定コメントをチェック！`
- `💬 チャンネル登録よろしくお願いします！`
