# YouTube Metadata Output Template

## タイトル構成

### 案A（感情訴求型）
```
「[印象的な歌詞の一節]」[曲名] / [歌手名] #shorts
```

### 案B（共感型）
```
[ターゲット層への呼びかけ]｜[曲名] - [歌手名] #shorts
```

## 概要欄構成

### シンプル版
```
🎵 [曲名] / [歌手名]

[テーマ要約（2-3行）]

#[曲名] #[歌手名] #リリックビデオ #[ジャンル] #shorts

▶ フル版は固定コメントをチェック！
```

## ハッシュタグ選定ガイド

| カテゴリ | 例 |
|---------|-----|
| 曲名 | #生きろ #ScarsIntoStars |
| 歌手名 | #Rin #SunoAI |
| 形式 | #リリックビデオ #lyricvideo |
| ジャンル | #邦楽 #応援ソング #バラード |
| プラットフォーム | #shorts |

**ルール**: 5個前後に厳選。多すぎるとスパム判定リスク。

## 印象的フレーズの抽出ポイント

1. **サビ部分** - 最も印象に残るフレーズ
2. **ラスト** - 締めくくりのメッセージ
3. **繰り返し** - 複数回登場するキーワード
4. **感情的** - 共感を呼ぶ表現
