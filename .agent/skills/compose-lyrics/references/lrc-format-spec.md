# LRC フォーマット仕様

## 基本フォーマット

```
[mm:ss.xx]歌詞テキスト
```

- `mm`: 分（2桁）
- `ss`: 秒（2桁）
- `xx`: 1/100秒（2桁）

## ヘッダータグ（推奨）

```lrc
[ti:タイトル]
[ar:アーティスト]
[al:アルバム]
[by:作詞者]
[offset:0]
```

## 仮タイムコード生成ルール

BPM不明時の目安間隔:

| セクション | 行あたり秒数 |
|-----------|------------|
| Verse | 3.0〜4.0秒 |
| Pre-Chorus | 2.5〜3.5秒 |
| Chorus | 3.0〜4.0秒 |
| Bridge | 3.0〜4.0秒 |
| Outro | 3.0〜5.0秒 |

セクション間の空白時間: 1〜2秒

## タイムコード計算手順

1. イントロを5〜8秒として開始
2. 各セクションの行数 × 行あたり秒数で長さを計算
3. セクション間に1〜2秒の空白を挿入
4. 空行（インスト区間）は `[mm:ss.xx]` だけの行（テキストなし）で表現

## サンプル

```lrc
[ti:ありがとう]
[ar:Suno AI]
[offset:0]
[00:00.00]
[00:08.00]有名なヒーローは
[00:11.50]誰かのために 命を懸ける

[00:22.00]

[00:26.00]君が僕にとって 大切な人だってことは
[00:30.00]誰にも負けないくらい 分かっているんだ
[00:34.00]

[00:41.50]ありがとう

```

## create_lyric_video スキルとの互換性

- `create_lyric_video` および `create_vertical_lyric_video` スキルは `.lrc` ファイルを入力として受け取る
- 生成した LRC ファイルを `public/suno_PJ/new/` に配置すれば、そのまま引き渡し可能
- タイムコードは音源ファイル入手後に調整を推奨
- `videoSource` は `public/suno_PJ/done/` を参照するため、音源は同じ名前で配置する
