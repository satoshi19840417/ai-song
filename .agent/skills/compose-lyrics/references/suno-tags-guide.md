# Suno タグリファレンス

## 使用可能なセクションタグ

| タグ | 日本語名 | 使用場面 |
|-----|---------|---------|
| `[Intro]` | イントロ | 曲の始まり（インスト可） |
| `[Verse]` | Aメロ | 番号なしでも可 |
| `[Verse 1]` | Aメロ1 | 推奨形式 |
| `[Verse 2]` | Aメロ2 | 推奨形式 |
| `[Pre-Chorus]` | Bメロ | サビ前 |
| `[Chorus]` | サビ | メインメロディ |
| `[Bridge]` | ブリッジ/Cメロ | 展開・転換部 |
| `[Outro]` | アウトロ | 曲の終わり |
| `[Instrumental]` | 間奏 | 演奏のみ |
| `[Guitar Solo]` | ギターソロ | ロック系に使用 |

## 書き方ルール

1. タグは `[タグ名]` の形式で単独行に記述
2. タグの後に空行不要（直後から歌詞を記述）
3. セクション間は空行1行で区切る
4. 同じタグを複数使用する場合は番号を付ける（例: `[Verse 1]`, `[Verse 2]`）

## 例

```
[Verse 1]
春の風が吹く日に
君の声が聞こえた
あの頃の記憶が
胸の中で蘇る

[Pre-Chorus]
もう一度だけ
会いたくて

[Chorus]
忘れられない
あの笑顔が
今も心に
輝いている

[Verse 2]
雨が窓を叩く夜
君との約束を思い出す
遠い空の向こうに
届けたい言葉がある

[Pre-Chorus]
もう一度だけ
触れたくて

[Chorus]
忘れられない
あの笑顔が
今も心に
輝いている

[Bridge]
時が流れても
変わらないものがある

[Chorus]
忘れられない
あの笑顔が
永遠に心に
輝いている

[Outro]
あの日みたいに
また笑えるから
```

## 注意事項

- タグは英語のみ（日本語タグは無効）
- Suno はタグを参考に曲構成を自動判定するが、必ず指定通りになるとは限らない
- `[Chorus]` タグのセクションが自動的にサビとして認識される
- 長すぎる歌詞はSunoが途中でカットする場合がある（1曲あたり目安400〜600文字）
