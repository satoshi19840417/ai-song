# 曲構成パターン集

## 標準 J-Pop 構成（2番構成）

```
[Intro]
[Verse 1]    ← Aメロ1
[Pre-Chorus] ← Bメロ1
[Chorus]     ← サビ1
[Verse 2]    ← Aメロ2
[Pre-Chorus] ← Bメロ2
[Chorus]     ← サビ2
[Bridge]     ← Cメロ
[Chorus]     ← サビ3（転調・盛り上げ）
[Outro]
```

行数目安:
- Intro: 0〜2行（インスト可）
- Verse: 4〜6行
- Pre-Chorus: 2〜4行
- Chorus: 4〜6行（繰り返し可）
- Bridge: 2〜4行（静かに始め最高潮へ）
- Outro: 1〜3行

---

## 短め構成（ショート動画向け）

```
[Verse 1]
[Chorus]
[Verse 2]
[Chorus]
[Outro]
```

2〜3分以内を想定。Pre-ChorusとBridgeを省略。

---

## バラード構成（ゆったり）

```
[Verse 1]
[Chorus]
[Verse 2]
[Chorus]
[Bridge]
[Chorus]
[Outro]
```

Pre-Chorusを省略してゆったりとした流れを強調。

---

## ロック構成（盛り上がり重視）

```
[Intro]
[Verse 1]
[Pre-Chorus]
[Chorus]
[Verse 2]
[Pre-Chorus]
[Chorus]
[Instrumental]
[Bridge]
[Chorus]
[Outro]
```

間奏（Instrumental）を入れてギターソロなどの演奏パートを確保。

---

## 各セクションの役割

| セクション | 役割 | 文字数目安 |
|-----------|------|-----------|
| Verse | 状況説明・心情描写 | 80〜120文字 |
| Pre-Chorus | サビへの助走・感情の高まり | 40〜80文字 |
| Chorus | 曲のメッセージ核心・繰り返し | 80〜120文字 |
| Bridge | 転換・視点変化 | 40〜80文字 |
| Outro | 締め・余韻 | 20〜60文字 |

---

## 選択ガイド

| ユーザーの希望 | 推奨構成 |
|--------------|---------|
| ショート動画向け | 短め構成 |
| バラード・感動系 | バラード構成 |
| ロック・激しい系 | ロック構成 |
| 標準（特に指定なし） | 標準J-Pop構成 |
