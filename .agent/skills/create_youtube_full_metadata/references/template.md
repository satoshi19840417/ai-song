## ハッシュタグ選定ガイド

| カテゴリ | 例 |
|---------|-----|
| 曲名 | #生きろ #ScarsIntoStars |
| 歌手名 | #Rin #SunoAI |
| 形式 | #リリックビデオ #lyricvideo #MV |
| ジャンル | #邦楽 #応援ソング #バラード #Jpop |
| 一般 | #音楽 #オリジナル曲 #新曲 |

**ルール**: 10個程度。ショート動画より多めでOK。

---

## チャプター生成のコツ

`.lrc`ファイルのタイムスタンプから：
1. **イントロ**: 最初の歌詞が始まる前
2. **Aメロ/Bメロ**: 最初の歌詞ブロック
3. **サビ**: 繰り返しフレーズや感情的なピーク
4. **間奏**: 歌詞のない区間
5. **アウトロ**: 最後の歌詞以降
