# YouTube Full Video Metadata Skill

YouTubeフル動画用のタイトル・概要欄を生成するスキル。

## Workflow

1. **入力情報の収集**
   - 歌詞ファイル（`.lrc`）のパス
   - 曲名
   - 歌手名

2. **歌詞ファイルの解析**
   - `.lrc`ファイルを読み込み
   - 印象的なフレーズを抽出
   - タイムスタンプからチャプター情報を生成

3. **タイトル生成（2案）**
   - **案A（スタンダード型）**: 【MV】曲名 / 歌手名 - リリックビデオ
   - **案B（検索最適化型）**: 曲名 - 歌手名 | Official Lyric Video

4. **概要欄生成（2パターン）**
   - **シンプル版**: 曲名、テーマ要約、歌詞全文、ハッシュタグ、CTA
   - **詳細版**: 曲名、テーマ詳細、チャプター、歌詞全文、クレジット、ハッシュタグ、CTA

5. **出力**
   - マークダウンファイルとして `[曲名]_YouTubeフル概要.md` を出力
   - 出力先: 動画ファイルと同じディレクトリ

## フル動画 vs ショート動画の違い

| 項目 | ショート動画 | フル動画 |
|------|-------------|----------|
| タイトル | #shorts必須 | 【MV】やOfficial表記 |
| 概要欄の長さ | 短め（5行程度） | 詳細（歌詞全文含む） |
| チャプター | 不要 | 推奨 |
| 歌詞掲載 | 抜粋のみ | 全文掲載 |
| ハッシュタグ | 5個 | 10個程度 |
| CTA | 簡潔に1行 | 複数のリンク可 |

## Best Practices

### タイトル
- 【MV】【Lyric Video】などの表記で動画タイプを明示
- 曲名と歌手名を明確に記載
- 検索されやすいキーワードを含める

### 概要欄構成
1. **冒頭3行**: スクロールなしで見える部分（最も重要）
2. **チャプター**: タイムスタンプ付きで曲構成を表示
3. **歌詞全文**: コピー可能な形式で掲載
4. **クレジット**: 作詞・作曲・歌手情報
5. **リンク集**: SNS、ストリーミング等
6. **ハッシュタグ**: 10個程度

### チャプター形式
```
0:00 イントロ
0:30 1番
1:20 サビ
2:10 2番
3:00 大サビ
3:45 アウトロ
```
