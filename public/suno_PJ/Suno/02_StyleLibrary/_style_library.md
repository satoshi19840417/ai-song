## よく使うスタイル文の例

### バラード系
```
Emotional J-Pop Ballad, Piano, Strings, Female Vocal, Melancholic
```

### ロック系
```
Energetic J-Rock, Electric Guitar, Powerful Drums, Male Vocal
```

### エレクトロ系
```
Electronic, Synth-heavy, Cool, Modern Beat
```

---

## 使い方
1. このファイルを参考にスタイル文を組み立てる
2. 各曲ファイルの `## 1) Style` セクションに記入
3. `## 3) Suno貼り付け用` にコピペ用として整形
